搭建博客hexo+github

2018-10-01 hexo 0 评论 字数统计: 689(字) 阅读时长: 2(分)

下面我们讲一下博客的搭建工作
现在写博客已经不是一种稀奇了，越来越多的人员加入到这个队伍，写属于自己的个人博客
博客根据学科的分类也有所不同，我这里主要讲一下程序的小类
手法是在网上摘抄的，费了老大劲儿头才在这网络中有了属于自己的一个网页
现在博客大部分都采用集成博客，即不需要自己设计，在网站上注册账号就可以了，如流行的CSDN等
但那种是不能用域名和主机IP地址访问的，据我目前所知能用域名访问的有两种方法
一个是服务器上基于PHP环境的WordPress，一个是github+hexo
都是采用主题样式的（即不需要自己设计，采用别人的主题直接用就可以）
因为前辈们准备的已经很清楚了，所以我就不自己再写一遍了，直接发链接好了
1.1 使用Hexo+Github一步步 搭建属于自己的博客[基础]
(https://www.cnblogs.com/fengxiongZz/p/7707219.html)
1.2 使用Hexo+Github一步步 搭建属于自己的博客 [进阶]
(http://www.cnblogs.com/fengxiongZz/p/7707568.html)
2.1 基于Hexo+Github+Coding搭建个人博客–基础篇[从菜鸟到放弃]
(http://yangbingdong.com/2017/build-blog-hexo-base/)

以上是两位先生的搭建思路，很详细，请大家参考，下面我讲一下我这篇主题

3.1 Cofess先生发布的hexo主题pure[模板]
(https://blog.cofess.com/)
3.2 主题博客github[主题很多，可以都点开看一下]
(https://github.com/cofess/)

下面看一下我对这篇主题的修改
4.1 为Hexo博客添加LiveRe评论系统[添加评论]
(https://www.imooc.com/article/details/id/22866)
注意在本主题pure中不需要那么多步骤，但需要申请出LiveRe账号
因为主题中已经给出了LiveRe代码，在\boke\themes\pure\layout_script_comment\livere.ejs中已经给出
只需在\pure_config.yml中Comment下选择
type: livere ,并给出livere: uid: —–即可，uid在自己申请的livere代码中查找
5.1 hexo博客的背景设置[背景设置]
(https://blog.csdn.net/com_ma/article/details/76039859)
5.2主题如何添加动态背景[即线条动态]
(http://shenzekun.cn/hexo%E5%A6%82%E4%BD%95%E6%B7%BB%E5%8A%A0%E5%8A%A8%E6%80%81%E8%83%8C%E6%99%AF.html)
在这里要注意下_layout.swig在本主题pure中地址为
boke(自己博客的地址)\themes\pure\layout\layout.ejs
6.1 Hexo博客主题pure修改记录[pure主题]
(https://blog.csdn.net/a945372160/article/details/78806700)

好了，经过以上步骤就可以有我的博客地址上的简单样式了，大家试一下吧
